# arcp — Ruby SDK for the Agent Runtime Control Protocol

Reference Ruby implementation of [ARCP](RFC-0001-v2.md) v1.0.

> **Status:** v0.1 in development. See [PLAN.md](PLAN.md) and
> [CONFORMANCE.md](CONFORMANCE.md) for what is implemented and what is
> deferred.

## Requirements

- Ruby 3.3 or newer
- Bundler 2.x

## Quickstart

```sh
bundle install
bundle exec rake
```

Detailed quickstart, sample walk-throughs, and architecture notes are
added during Phase 7.
