# Conformance

The full spec-to-code matrix lives at the repository root:

- [`CONFORMANCE.md`](../CONFORMANCE.md)

It maps every numbered spec section (§4–§16) to the Ruby class or method
that implements it, and lists each `MUST`/`SHOULD`/`MAY` assertion with
its implementation status (`PASS`, `N/A`, or a note).
